# 🌎 Hello World - Landing Page

Projeto desenvolvido com HTML5 e CSS3 com o objetivo de praticar a criação de uma landing page moderna sem o auxílio de ferramentas de design como Figma. A página apresenta um layout simples, agradável e focado em estudos de estruturação, posicionamento e estilização de elementos.

---

## 🎯 Objetivo

O principal objetivo deste projeto é colocar em prática conceitos fundamentais de desenvolvimento Front-End, criando uma interface visual utilizando apenas HTML e CSS.

---

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- Google Fonts (Poppins e Inter)

---

## ✨ Funcionalidades

- Menu de navegação horizontal
- Seção principal com chamada de destaque
- Botões de ação
- Imagem ilustrativa
- Layout moderno com Flexbox
- Fundo com gradiente

---

## 🖥️ Layout

A página é composta por:

### Header
- Logo/Título "Hello Word"
- Menu de navegação
  - Home
  - Sobre
  - Blog
  - Suporte
  - Contato

### Conteúdo Principal
- Título de destaque
- Subtítulo explicativo
- Texto de apoio
- Botão principal
- Botão secundário

### Ilustração
- Imagem de um planeta posicionada ao lado do conteúdo principal.
